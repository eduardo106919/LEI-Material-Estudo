# Algoritmos e Complexidade

## Estratégias de Estudo

Algoritmos e Complexidade é reconhecida como uma das disciplinas mais desafiantes do 2º ano, 1º semestre. Para ter sucesso, é fundamental investir tempo na compreensão dos conceitos teóricos e, sobretudo, na prática intensiva de exercícios. Recomenda-se:

- Resolver muitos exercícios práticos, especialmente problemas de análise de complexidade e implementação de algoritmos e estruturas de dados lecionadas.
- Rever exemplos resolvidos nas aulas e tentar resolvê-los novamente sem consultar a solução.
- Não deixar acumular matéria: a disciplina é cumulativa e os tópicos estão interligados.

A prática constante e a revisão ativa são essenciais para consolidar o conhecimento e ganhar confiança para os testes e exames.
